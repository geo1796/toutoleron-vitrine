# Vitrine Toutoleron - project guidelines

This document provides essential guidance for contributing to the project.

## Project overview

Vitrine Toutoleron is a simple static website to showcase the Toutoleron company capabilities and services.

## Tech stack

- Astro - Daisyui - TailwindCSS - Vite

## Project Structure

- Homepage mockup available at project root `TOUTOLERON_Homepage.pdf`
- Assets folder placed in `src/assets` contains ui elements from the mockup
- Css styles placed in `src/assets/app.css`
